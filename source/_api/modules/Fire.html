attr

  • propertyName
  • [attributes ]
)
object | Undefined utils/api/engine-framework/core/attribute.js:7

Tag the class with any meta attributes, then return all current attributes assigned to it. This function holds only the attributes, not their implementations.

Name Type Description
constructor function | object

the class or instance. If instance, the attribute will be dynamic and only available for the specified instance.
propertyName string

the name of property or function, used to retrieve the attributes
attributes optional object

the attribute table to mark, new attributes will merged with existed attributes. Attribute whose key starts with '_' will be ignored.

Returns:

Type: object | Undefined

return all attributes associated with the property. if none undefined will be returned

Example:

 var myClass = function () { this.value = 0.5 };
 Fire.attr(myClass, 'value');         // return undefined
 Fire.attr(myClass, 'value', {}).min = 0;  // assign new attribute table
             //associated with 'value', and set its min = 0
 Fire.attr(myClass, 'value', {       // set values max and default
    max: 1,
    default: 0.5,
 });
 Fire.attr(myClass, 'value');  // return { default: 0.5, min: 0, max: 1 }
