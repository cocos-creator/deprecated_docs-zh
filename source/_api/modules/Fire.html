Object | Undefined

定义于 utils/api/core/attribute.js:4

Tag the class with any meta attributes, then return all current attributes assigned to it. This function holds only the attributes, not their implementations.

参数列表:

  • constructor Function | Object
    the class or instance. If instance, the attribute will be dynamic and only available for the specified instance.
  • propertyName String
    the name of property or function, used to retrieve the attributes
  • attributes Object optional
    the attribute table to mark, new attributes will merged with existed attributes. Attribute whose key starts with '_' will be ignored.

返回值:

Object | Undefined: return all attributes associated with the property. if none undefined will be returned

例子:

var klass = function () { this.value = 0.5 }; Fire.attr(klass, 'value'); // return undefined Fire.attr(klass, 'value', {}).min = 0; // assign new attribute table associated with 'value', and set its min = 0 Fire.attr(klass, 'value', { // set values max and default max: 1, default: 0.5, }); Fire.attr(klass, 'value'); // return { default: 0.5, min: 0, max: 1 }
