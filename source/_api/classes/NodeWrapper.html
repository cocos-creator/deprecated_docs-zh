Class NodeWrapper

Defined in: utils/api/engine-framework/runtime/wrappers/node.js:23
Module: Fire.Runtime

: This is a wrapper class for operating node with editor script The instance of this class is a wrapper, not a node. You can use Fire(node) to get the wrapper if you really want to use these API on runtime nodes. Note: API that has a suffix "N" return Runtime's native Node type
